# 快速开始 - EatWhat

## 3步开始使用

### 第1步：打开应用

双击 `index.html` 在浏览器中打开，或者部署后访问您的 GitHub Pages 网址。

### 第2步：获取 Gemini API 密钥

1. 访问：https://makersuite.google.com/app/apikey
2. 使用 Google 账号登录
3. 点击"创建 API 密钥"
4. 复制密钥

### 第3步：输入 API 密钥并开始分析！

1. 在提示框中粘贴您的 API 密钥
2. 上传食物照片
3. 点击"分析食物"
4. 立即查看结果！

## 获得最佳效果的建议

📸 **拍照技巧：**
- 光线充足
- 清晰展示食物
- 避免模糊图片
- 展示整盘食物

🍔 **适用场景：**
- 餐厅餐点
- 家常菜
- 盘中多种食物
- 零食和甜点

⚡ **快捷操作：**
- 直接拖拽照片
- 移动设备可使用相机
- 保存餐点以追踪历史
- 查看每日/每周统计数据

## 需要帮助？

- 查看 README.md 了解故障排除
- 确认 API 密钥正确
- 确保图片小于 10MB
